### Launcher 2

Experimental Launcher for [Avalon](https://github.com/getavalon).

> NOT READY FOR USE

<br>

**What is it?**

It's a pure-Python GUI for the Avalon Launcher, that runs in your browser with no external dependencies and supports Python 2 and 3.
